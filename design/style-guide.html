Chicken Road — Art Direction Style Guide

Direction: "Arcade Pixel." Chunky, high-saturation 16×16-grid pixel art with hard offset shadows, notched panel corners and zero anti-aliasing, wrapped in a constant dark arcade-cabinet UI chrome so the product identity holds steady while the world transforms across 3 stages. Grounded via the ui-ux-pro-max design-system lookup (Pixel Art style, Press Start 2P/VT323 mood reference, Arcade & Retro Game color system).

This file is fully self-contained (no external network requests, no image files, no CDN fonts) — every sprite and screen below is drawn with inline <canvas> pixel-fills and CSS, exactly matching the constraint the shipped single-file game must also honor. Open directly in a browser.

Full spec, rationale, animation timings and accessibility notes: see docs/DESIGN.md.

2 — Sprite Showcase

All sprites are drawn on a native pixel grid with imageSmoothingEnabled = false, then scaled up crisply. Animation demo speeds below are intentionally slowed for reviewability — real in-game timings are noted per card (see also DESIGN.md §1.7 / §4).

Chicken — Idle (head-bob)
4-frame bob loop, 180ms/frame (720ms cycle) in-game. Demo slowed slightly for visibility.
Chicken — Hop (squash & stretch)
Real hop = 140–160ms total. Demo cycle slowed to ~900ms for readability.
Vehicle — Car (Stage 1)
Headlight glow is the one deliberate soft-edge exception in the art system.
Vehicle — Truck (Stage 3)
Darker cab block + longer cargo body; harsher industrial headlight glow.
River — Log & Lily Pad
Both bob ±1px on a slow 1.2s float cycle.
Corn Pickup — Spin
4-frame horizontal squash fakes a Y-axis spin, 480ms/loop in-game.
Train + Warning Signal
Trailing ghost copies sell speed; signal blinks red every 200ms during telegraph.
Eagle (fall-behind)
Spread-wing silhouette, scales 40%→100% on approach during swoop-grab.
